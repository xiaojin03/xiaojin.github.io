博客教程---hexo

搭建个人博客教程—hexo

视频教程连接：程序羊教程

博客教程辅助：CNDN博客教程

一、如何部署环境

准备

  • Hexo：Hexo快速、简洁且高效的博客框架
  • Node.js：建立在Chrome上的JavaScript运行引擎
  • Git：一款免费、开源的分布式版本控制系统
  • GitHub：国内一款面向开发者的云端开发平台，提供代码托管，运行空间，质量控制，项目管理等功能

下载Node.js、Git软件，注册GitHub账号

下载Hexo

  1. Win+R打开运行窗口，输入cmd打开交互界面

  2. 可以先分别输入命令查看node和npm的版本号

-> node -v

-> npm -v

  1. 利用npm来下载hexo，由于hexo来自国外，直接下载速度很慢，需要借用淘宝的镜像语言下载来
  • 先利用npm下载cnpm：

-> npm install -g cnpm –registry=https://registry.npm.taobao.org

  • 在利用cnpm来下载hexo

-> cnpm install -g hexo-cli

二、如何初始化个人博客

用Hexo建立博客

  • 先查看当前操作路径

-> chdir

  • 更改路径位置

-> d:

  • 创建名为blog的文件夹

-> mkdir blog

  • 进入blog文件夹

-> cd blog/

  • 初始化一个博客

-> hexo init

  • 启动博客

-> hexo s

或-> hexo server

在本地端输入网址localhost:4000就可以访问博客了

window交互的命令行 操作
chair(pwd) 查看当前文件路径
dir(ls) 查看当前文件夹中的文件
cd source/_posts/ 按路径进入文件
cd ../.. 退回前两层的路径
ls-l(ll) 修改

三、如何写博客

  • 新建一个名为“我的第一个博客”的博客

-> hexo new “我的第一个博客”

交互窗口会显示
INFO Created: D:\blog\source_posts\我的第一篇博客.md

  • 然后可以用typora等软件来编辑此文件
  • 回到 D:\blog\的位置清理博客，重新生成

-> hexo clean

-> hexo generate

四、如何把个人博客部署到远端

免费的方法：将博客部署到GitHub上公开使用

注意：用户部署个人博客的GitHub仓库的命名必须符合特定要求才行

规范：账号名.github.io

  • 安装一个GitHub的插件：-> cnpm install –save hexo-deployer-git

  • 接着对blog文件中的_config.yml文件进行修改配置

注意：每个冒号后面要有一个空格！

拉到文件最下面进行以下设置：

type: git
repo: https://github.com/xiaojin03/xiaojin.github.io.git
branch: master

  • 回到交互界面对hexo进行部署

-> hexo deploy

然后按提示输入GitHub的账号和密码

  • 成功后按仓库名来访问就可
